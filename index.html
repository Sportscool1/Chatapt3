Creating a chatbot in HTML involves using HTML for the structure and CSS for styling, along with JavaScript for the interactive chat functionality. Here's a simple example to get you started:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Chatbot Example</title>
<style>
.chat-container {
  width: 300px;
  height: 400px;
  border: 1px solid #ccc;
  overflow: auto;
  padding: 10px;
}
.user-message {
  background-color: #f0f0f0;
  border-radius: 5px;
  padding: 5px;
  margin: 5px 0;
}
.bot-message {
  background-color: #e0e0e0;
  border-radius: 5px;
  padding: 5px;
  margin: 5px 0;
}
</style>
</head>
<body>
<div class="chat-container" id="chat-container">
  <div class="bot-message">Hello! How can I help you today?</div>
</div>
<input type="text" id="user-input" placeholder="Type your message...">
<button onclick="sendMessage()">Send</button>

<script>
function sendMessage() {
  const userInput = document.getElementById("user-input").value;
  const chatContainer = document.getElementById("chat-container");

  const userMessage = document.createElement("div");
  userMessage.className = "user-message";
  userMessage.textContent = userInput;
  chatContainer.appendChild(userMessage);

  // You can add logic here to generate bot responses based on user input
  const botMessage = document.createElement("div");
  botMessage.className = "bot-message";
  botMessage.textContent = "I'm a chatbot! How can I assist you?";
  chatContainer.appendChild(botMessage);

  document.getElementById("user-input").value = "";
}
</script>
</body>
</html>

This code provides a basic chatbot interface where users can input messages and receive bot responses. You can further enhance this by adding more sophisticated logic and design elements. Feel free to customize it to meet your specific needs.
